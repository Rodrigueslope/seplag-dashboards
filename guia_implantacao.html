Guia de Implantação dos Dashboards SEPLAG-CE no GitHub Pages

Este documento fornece instruções detalhadas para implantar os dashboards da SEPLAG-CE no GitHub Pages e obter links permanentes para integração no tour virtual Matterport.

Nota: Este guia assume que você possui uma conta no GitHub. Se ainda não tiver uma, será necessário criar uma em github.com/join.

Passo 1: Criar um novo repositório no GitHub

  1. Acesse github.com/new
  2. Nomeie o repositório como seplag-dashboards
  3. Adicione uma descrição opcional (ex: "Dashboards interativos para o tour virtual da SEPLAG-CE")
  4. Mantenha o repositório como Público (necessário para GitHub Pages gratuito)
  5. Marque a opção "Add a README file"
  6. Clique em "Create repository"

Passo 2: Fazer upload dos arquivos HTML

  1. No seu repositório recém-criado, clique no botão "Add file" e selecione "Upload files"
  2. Arraste e solte os seguintes arquivos HTML ou use o seletor de arquivos para carregá-los:
    • index.html (página inicial)
    • clima.html (dashboard de condições climáticas)
    • ar.html (dashboard de qualidade do ar)
    • mobilidade.html (dashboard de mobilidade urbana)
    • energia.html (dashboard de consumo energético)
  3. Adicione uma mensagem de commit (ex: "Adicionar arquivos dos dashboards")
  4. Clique em "Commit changes"

Passo 3: Configurar o GitHub Pages

  1. No seu repositório, clique na aba "Settings"
  2. No menu lateral esquerdo, clique em "Pages"
  3. Na seção "Source", selecione "main" no dropdown "Branch"
  4. Clique em "Save"
  5. Aguarde alguns minutos para que o GitHub Pages publique seu site
  6. Após a publicação, você verá uma mensagem indicando que seu site está publicado em https://[seu-usuario].github.io/seplag-dashboards/

Passo 4: Verificar a implantação

  1. Acesse o link fornecido pelo GitHub Pages (https://[seu-usuario].github.io/seplag-dashboards/)
  2. Verifique se a página inicial carrega corretamente
  3. Teste cada um dos dashboards clicando nos botões "Acessar Dashboard"
  4. Verifique se as visualizações interativas funcionam corretamente
  5. Teste a navegação entre os dashboards e a página inicial

Passo 5: Obter os links para integração no Matterport

Após a implantação bem-sucedida, você terá os seguintes links permanentes para integração no tour virtual Matterport:

Links para os Dashboards:

  • Página Inicial: https://[seu-usuario].github.io/seplag-dashboards/
  • Dashboard de Condições Climáticas: https://[seu-usuario].github.io/seplag-dashboards/clima.html
  • Dashboard de Qualidade do Ar: https://[seu-usuario].github.io/seplag-dashboards/ar.html
  • Dashboard de Mobilidade Urbana: https://[seu-usuario].github.io/seplag-dashboards/mobilidade.html
  • Dashboard de Consumo Energético: https://[seu-usuario].github.io/seplag-dashboards/energia.html

Passo 6: Integrar os links no tour virtual Matterport

  1. Acesse o editor do Matterport para o tour virtual da SEPLAG-CE
  2. Adicione Mattertags nos pontos estratégicos do tour
  3. Configure cada Mattertag para abrir o link do dashboard correspondente:
    • Selecione a opção "Link" ao configurar a Mattertag
    • Cole o URL do dashboard correspondente
    • Defina um título apropriado (ex: "Condições Climáticas")
    • Adicione uma descrição opcional
  4. Salve as alterações no tour virtual
  5. Teste a integração acessando o tour e clicando nas Mattertags
Importante: Os dashboards estão configurados com dados simulados. Para conectá-los a dados reais em tempo real, será necessário modificar os arquivos HTML para fazer requisições às APIs correspondentes (OpenWeatherMap, AQICN, etc.) e atualizar o repositório.

Manutenção e Atualização

Para atualizar os dashboards no futuro:

  1. Faça as alterações necessárias nos arquivos HTML
  2. Acesse seu repositório no GitHub
  3. Navegue até o arquivo que deseja atualizar
  4. Clique no ícone de lápis para editar o arquivo
  5. Faça as alterações necessárias
  6. Adicione uma mensagem de commit descrevendo as alterações
  7. Clique em "Commit changes"
  8. As alterações serão automaticamente publicadas no GitHub Pages após alguns minutos

Este guia foi preparado como parte do projeto de dashboards interativos para o tour virtual da SEPLAG-CE.
